Übung 1: HTML Grundstruktur

Aufgabe:

Erstelle ein neues HTML-Dokument mit der korrekten Grundstruktur, wie sie in der Lektion beschrieben wurde. Das Dokument sollte folgende Elemente enthalten:

  • Die HTML5-Dokumenttyp-Deklaration
  • Ein html-Element mit dem Sprachattribut für Englisch
  • Einen head-Bereich mit Titel "Document"
  • Einen body-Bereich mit einer Überschrift 1. Ordnung "My First Webpage"

Deine Lösung:

Tipp:

Denke an die Reihenfolge der Elemente und schau dir nochmal den Boilerplate-Code aus der Lektion an.

Übung 3: Erstellen einer index.html

Aufgabe:

Wie du in der Lektion gelernt hast, ist die index.html eine besonders wichtige Datei für jede Website. Erstelle eine vollständige index.html für eine persönliche Profilseite mit:

  • Einem aussagekräftigen Titel im head-Bereich
  • Einer Hauptüberschrift mit deinem Namen (oder einem fiktiven Namen)
  • Einem Absatz mit einer kurzen Beschreibung
  • Einer ungeordneten Liste mit 3 Interessen oder Fähigkeiten
  • Einem div-Element, das einen zweiten Absatz umschließt
  • Einem Button zum "Kontaktieren"

Deine Lösung:

Tipp:

Vergiss nicht die grundlegende HTML-Struktur mit DOCTYPE, html, head und body. Alle sichtbaren Inhalte müssen im body-Element stehen.

Übung 4: Ein kleines HTML-Projekt

Aufgabe:

Erstelle eine komplette Webseite zu einem Thema deiner Wahl (z.B. Haustiere, Filme, Sport, Reisen). Integriere alle gelernten HTML-Konzepte:

  • Die korrekte Grundstruktur (DOCTYPE, html, head, body)
  • Verschiedene Überschriften (h1, h2)
  • Mehrere Absätze mit Text
  • Mindestens eine ungeordnete Liste
  • Mindestens einen Link zu einer externen Webseite
  • Mindestens ein div-Element zur Gruppierung von Inhalten
  • Einen Button
  • Ein einfaches Formular mit einem Textfeld

Deine Lösung:

Tipp:

Plane deine Seite zuerst: Welches Thema möchtest du behandeln? Welche Inhalte willst du darstellen? Wie kannst du die verschiedenen HTML-Elemente sinnvoll einsetzen?

Übung 5: Fehler finden und beheben

Aufgabe:

Im folgenden HTML-Code haben sich einige Fehler eingeschlichen. Finde und korrigiere sie:


<!DOCTYPE html>
<html lang="en"
  <head>
    <title>Fehlerhafte Seite<title>
  </head>
  <body>
    <h1>Willkommen auf meiner Webseite</h1
    <p>Dies ist ein Absatz mit <strong>hervorgehobenem</string> Text.</p>
    <ul>
      <li>Erster Punkt</li>
      <li>Zweiter Punkt<li>
      <li>Dritter Punkt</li>
    </ul>
    <a href="https://www.example.com">Besuche Example<a>
    <div>
      <p>Ein Absatz in einem Container.</p>
    <div>
    <button>Klick mich!
  </body>
</html

Deine Lösung:

Tipp:

Achte besonders auf fehlende Klammern, nicht geschlossene Tags und falsch geschriebene Element-Namen.

Bonus-Herausforderung: Mini-Website

Aufgabe:

Erstelle eine kleine Website mit mehreren verlinkten Seiten. Deine Website sollte Folgendes enthalten:

  • Eine index.html als Startseite
  • Mindestens zwei weitere HTML-Seiten
  • Navigation zwischen allen Seiten (Links)
  • Alle wichtigen HTML-Elemente, die du gelernt hast
  • Eine strukturierte und semantisch sinnvolle Gestaltung

Diese Aufgabe ist nicht interaktiv überprüfbar - erstelle die Dateien auf deinem Computer und teste sie in deinem Browser!

Tipp:

Du kannst für jede Seite eine eigene HTML-Datei erstellen und sie mit relativen Links verknüpfen, z.B. <a href="seite2.html">Zur Seite 2</a>.
