What is HTML?

HTML stands for Hypertext Markup Language and is the fundamental building block from which websites are created.

HTML allows us to structure a website through sections, paragraphs, and headings. It enables us to create text, links, maps, lists, buttons, and much more.

1. HTML Tags

The structure of an HTML file is very simple. Almost all elements are merely content enclosed in opening and closing HTML tags. For example, you would write the title of a webpage like this:

<title>Document</title>

Although the current HTML version (HTML5) contains 140 different tags, most websites are created using only a fraction of them. You have already seen the title tag, let's take a quick look at other commonly used tags. You will probably use these in your project as well:

  • <html> - Must enclose all the remaining code
  • <head> - Contains the title, the link to the CSS file (styling) and various other information
  • <body> - Contains the visible content of a website
  • <p> - Paragraph, simple text goes in here
  • <h1> - Top-level heading of a website
  • <div> - Structure that groups content
  • <ul> - Unordered list, containing list elements <li>
  • <a> - Link tag, used to link from one website to another
  • <img> - An image in an HTML page
  • <form> - For creating an HTML form for user input
  • <button> - Creates a clickable button

2. HTML Structure

Now that you know the majority of relevant tags, the only question remaining is how we actually write HTML code. For this, it's important to know that all HTML documents have the same boilerplate-like structure that must be present before anything useful can be done. The boilerplate looks something like this and is often simply accessible via a shortcut (so you don't have to retype everything every time you create a new file):


<!DOCTYPE html>
<html lang="en">
  <head>
    <title>Document</title>
  </head>
  <body>
  </body>
</html>

All content that appears on a webpage is written between the <body> tags. Now we know the basic structure and some HTML tags, but how and where do we write HTML code? The answer is simple: We can write HTML code anywhere! Even in a simple text editor, as long as we save the code as .html, we can view the results in a web browser.

Please note that we should always save the home page of our websites as index.html. This is because web servers look for an index.html page by default when users access our websites - and if there isn't one, it can lead to major problems.
