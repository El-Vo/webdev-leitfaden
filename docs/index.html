Was ist HTML?

HTML steht für Hypertext Markup Language und ist der Grundbaustein, aus dem Webseiten erstellt werden.

HTML ermöglicht es uns, eine Webseite zu strukturieren durch Abschnitte, Absätze und Überschriften. Es erlaubt uns, Text, Links, Karten, Listen, Buttons und vieles mehr zu erstellen.

1. HTML Tags

Die Struktur einer HTML-Datei ist sehr einfach. Fast alle Elemente sind lediglich Inhalte, die in öffnenden und schließenden HTML-Tags eingeschlossen sind. Zum Beispiel würdest du den Titel einer Webseite so schreiben:

<title>Document</title>

Obwohl die aktuelle HTML-Version (HTML5) 140 verschiedene Tags enthält, werden die meisten Webseiten nur mit einem Bruchteil davon erstellt. Du hast bereits das title-Tag gesehen, lass uns einen schnellen Blick auf weitere häufig verwendete Tags werfen. Diese wirst du wahrscheinlich auch in deinem Projekt verwenden:

  • <html> - Muss den gesamten restlichen Code umschließen
  • <head> - Enthält den Titel, den Link zur CSS-Datei (Styling) und verschiedene andere Informationen
  • <body> - Enthält den sichtbaren Inhalt einer Website
  • <p> - Absatz, einfacher Text gehört hier hinein
  • <h1> - Überschrift der obersten Ebene einer Website
  • <div> - Struktur, die Inhalte gruppiert
  • <ul> - Ungeordnete Liste, die Listenelemente <li> enthält
  • <a> - Link-Tag, wird verwendet, um von einer Website zur nächsten zu verlinken
  • <img> - Ein Bild in einer HTML-Seite
  • <form> - Zum Erstellen eines HTML-Formulars für Benutzereingaben
  • <button> - Erstellt einen anklickbaren Button

2. HTML Struktur

Nachdem du nun die Mehrheit der relevanten Tags kennst, bleibt nur noch die Frage, wie wir tatsächlich HTML-Code schreiben. Dafür ist es wichtig zu wissen, dass alle HTML-Dokumente die gleiche Boilerplate-ähnliche Struktur haben, die vorhanden sein muss, bevor etwas Nützliches getan werden kann. Der Boilerplate sieht etwa so aus und ist oft einfach über eine Verknüpfung zugänglich (sodass du nicht jedes Mal alles neu tippen musst, wenn du eine neue Datei erstellst):


<!DOCTYPE html>
<html lang="en">
  <head>
    <title>Document</title>
  </head>
  <body>
  </body>
</html>

Der gesamte Inhalt, der auf einer Webseite erscheint, wird zwischen den <body>-Tags geschrieben. Jetzt kennen wir die Grundstruktur und einige HTML-Tags, aber wie und wo schreiben wir HTML-Code? Die Antwort ist einfach: Wir können HTML-Code überall schreiben! Sogar in einem einfachen Texteditor, solange wir den Code als .html speichern, können wir die Ergebnisse im Webbrowser ansehen.

Bitte beachte, dass wir die Startseite unserer Websites immer als index.html speichern sollten. Das liegt daran, dass Webserver standardmäßig nach einer index.html-Seite suchen, wenn Benutzer auf unsere Websites gelangen - und wenn es keine gibt, kann das zu großen Problemen führen.
